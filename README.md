# Loan Management API

A simple Spring Boot backend project demonstrating loan processing system.

## Features
- Create User
- Apply Loan
- Get User Loans

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## API Endpoints

POST /api/users  
POST /api/loans/{userId}  
GET /api/loans/{userId}  

## Run
mvn spring-boot:run
